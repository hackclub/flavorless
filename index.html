a hack club ysws where
ys: a (functional) website without any css
ws: a sticker sheet and prizes to the coolest submissions (ones that make my eyes bleed)

deadline

Submissions end on April 1st for individuals

Clubs can continue indefinitely

rules

  • no css allowed - not even inline styles
    • no <canvas> either
    • <tt> and images are allowed
    • deprecated html attributes that were replaced with css variants are allowed
    • svg is allowed but you can't use css attributes like style, fill, or stroke
  • javascript is allowed - use it creatively
  • your website has to be functionally complete - meaning it must have some use and have functionality but not necessarily pretty

requirements for rewards

  • 5+ unique html features to earn $3 per website 5+ html and 1+ javascript function to earn $5 per website
